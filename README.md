# java-web-maven-blade-api

## Description
A POC for blade framework rendering html page intercepted with middleware.

## Tech stack
- cors
- middleware
- endpoints
    - get, put, delete

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
